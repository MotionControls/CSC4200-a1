## Running

`appserver [address] [port]`

- *Optional*: `address` specifies an address for the server to use. It will use the first available localhost if unspecified.
- *Optional*: `port` specifies a port for the server to use. Defaults to 8008.

`appclient [address] [port]`
- `address` specifies the address to connect to.
- *Optional*: `port` specifies the port to connect to. Defaults to 8008.
